# Tutorial de Python para Ciencia de Datos, IA y Desarrollo

Este repositorio contiene una traducción completa al español de los apuntes y tutoriales del curso **Python for Data Science, AI & Development** ofrecido por IBM en la plataforma Coursera.

## Descripción

Este proyecto recopila y traduce todos los materiales del curso mencionado, organizados en módulos temáticos. El objetivo es proporcionar una referencia accesible en español para estudiantes y profesionales interesados en aprender Python aplicado a la ciencia de datos, inteligencia artificial y desarrollo de software.

Los apuntes cubren desde conceptos básicos de Python hasta temas avanzados como manipulación de datos con Pandas y NumPy, APIs, web scraping y manejo de diferentes formatos de archivo.

## Cómo Usar

1. **Navega por los módulos:** Cada carpeta representa un módulo del curso. Los archivos en inglés están en la carpeta `en/`, y las traducciones al español en la carpeta `es/`.

2. **Lee los apuntes:** Los archivos están en formato Markdown, ideales para visualizar en editores como VS Code, GitHub o cualquier lector de Markdown.

3. **Practica el código:** Ejecuta los ejemplos de código en tu entorno Python local. Se recomienda usar Jupyter Notebook para los ejercicios interactivos.
